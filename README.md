　(構成LINEBot+GAS+LIFF+GitHub)
入力は、LINE (SCCK @494fqxjs)
成績入力
　LINE入力(チームNo.、回戦、第１投者得点、第２投者得点、第３投者得点、第４投者得点、第５投者得点)
    例　(1,1,15,18,24,16,31)意味、チーム1の第１回戦は、15点,18点,24点,16点,31点
    改行 4,1,17,22,16,21,26 意味、チーム4の第１回戦は、17点,22点,16点,21点,26点
    複数行入力できる。入力後、確認して LINE 送信
    LINE から入力値と合計が返ってくる。
    返信を確認する。(二重チェック)グループラインなら、複数人で確認できる。
  選手交代（チームNo.、ポジション、名前）例１,３、前田健太（チーム1の3番が前田健太に交代します）
  順番変更（チームNo.、ポジション）例1，5，2，4，3，1（チーム1の順番が5，2，4，3，1の順番になります）
  スコア修正（訂正又は修正、チームNo.、回戦、正しい5つのスコア）
